Instructions for running Select Sounds Web Scraper.

1. Open terminal program 

2. Run following command: cd Desktop/select-sounds-web-scraper/web-scraper. This will move terminal to web scraper folder

2. Run following command: python3 web_scraper.py

3. Web Scraper will run in terminal

4. Enter discogs links when prompted and info will be entered into chosen excel file.

5. When want to exit, simply press enter without putting any data in.
